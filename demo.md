# vCon Viewer Extension - Demo Script 🎬

## 🎯 Demo Overview

This demo showcases the vCon Viewer extension's capabilities for viewing and interacting with IETF vCon files in VS Code.

## 🚀 Demo Flow

### 1. Introduction (30 seconds)
- **Welcome**: "Today I'm excited to show you the vCon Viewer extension for VS Code"
- **Problem**: "Voice conversation data is complex and hard to navigate in raw JSON format"
- **Solution**: "vCon Viewer transforms this data into an intuitive, interactive experience"

### 2. Installation & Setup (1 minute)
- Open VS Code
- Go to Extensions (Ctrl+Shift+X)
- Search for "vCon Viewer"
- Show the extension details and features
- Install the extension

### 3. Opening a vCon File (2 minutes)
- Open the `sample.vcon.json` file
- Point out the automatic detection notification
- Show the file structure in JSON format
- Highlight the complexity of raw vCon data

### 4. Using the vCon Viewer (3 minutes)
- Use Command Palette: "Open vCon Viewer"
- Show the rich viewer interface
- Navigate through different sections:
  - **Metadata**: Version, UUID, timestamps, subject
  - **Parties**: Participants with contact information
  - **Dialog**: Conversation content with audio, text, video
  - **Attachments**: Files and documents
  - **Analysis**: Transcripts, sentiment, intent
  - **Jumps**: Conversation flow markers
  - **Groups**: Participant groupings
  - **Signing**: Digital signatures
  - **Encryption**: Encryption metadata

### 5. Interactive Features (2 minutes)
- **Export Functionality**:
  - Export as JSON (opens in new tab)
  - Export as Text (formatted plain text)
- **Theme Integration**: Show how it adapts to VS Code themes
- **Responsive Design**: Demonstrate the clean, professional interface

### 6. Real-World Use Cases (2 minutes)
- **Contact Centers**: Call quality monitoring and compliance
- **Voice AI**: Training data analysis and model development
- **Research**: Conversation analysis and linguistics studies
- **Compliance**: Regulatory requirements and audit trails

### 7. Technical Highlights (1 minute)
- **IETF Standard**: Implements official vCon specification
- **Smart Detection**: Automatically identifies vCon files
- **Validation**: Ensures data integrity and format compliance
- **Performance**: Fast loading and smooth interaction

### 8. Conclusion & Call to Action (30 seconds)
- **Summary**: "vCon Viewer makes voice conversation data accessible and actionable"
- **Benefits**: "Save time, improve insights, and enhance productivity"
- **Next Steps**: "Install the extension and start exploring your vCon data today!"
